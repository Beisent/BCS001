# 颜色格式

​	由于计算机图形学与颜色紧密相关，可以说是计算机中的艺术。我们又知道计算机都是由数据组成的，所以我们为了表示颜色我们需要一种编码方式。接下来我会介绍计算机图形学中主要使用的颜色格式，以后我们通常使用这种格式。

## RGB和RGBA

​	RGB是由三个分量组成一个三元数组 (x,y,z)，其中第一个元素通道 x 表示红色占整体颜色的比重，其中第二个元素通道 y 表示绿色占整体颜色的比重，其中第三个元素通道 z 表示蓝色占整体颜色的比重。该模型基于人眼对这三种基本色光的感知机制，通过不同强度的红、绿、蓝三原色光叠加，可以合成出人眼可见的绝大多数颜色。

​	在计算机中，每个颜色通道通常用一个 0 到 255 的整数表示即 8 位，因此一个完整的 RGB 颜色可以用三个字节来存储，例如 (255, 0, 0) 表示纯红色，(0, 255, 0) 表示纯绿色，而 (255, 255, 255) 则是白色。

GBA 是 RGB 的扩展形式，其中的 “A” 代表 Alpha 通道，用于表示颜色的透明度。Alpha 值同样通常取 0 到 255 之间的整数，0 代表完全透明，255 代表完全不透明。
